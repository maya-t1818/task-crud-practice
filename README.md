# task-crud-practice

## 概要
COACHTECH 教材 Tutorial 11-2「タスク管理APIのCRUD実装」で作成した成果物です。


## 使用技術
- PHP 8.x
- Laravel 10.x
- REST API（CRUD）
- API Resources、FormRequest（バリデーション）

## 学んだこと
- sail artisan make:controller <controller名>　--apiで作成
- HTTPステータスコード使い分け
- apiResourceで一括ルーティング

## 動作確認
Postmanで各エンドポイントにリクエストを送信
